# Help Skill

Display available skills and how to use them.

## Usage

```
/help          - Show all skills
/help ralph    - Show ralph skill details
omk help       - Show CLI help
```

## Available Skills

- `$ralph` - Persistent completion loop
- `$team` - Coordinated multi-agent execution
- `$plan` - Create implementation plan
- `$deep-interview` - Socratic requirements clarification
- `$autopilot` - Full autonomous pipeline
- `$cancel` - Cancel active modes

## Quick Start

1. Run `omk setup` to initialize
2. Run `omk --high` to start with high reasoning
3. Type `$ralph "your task"` to begin
