# Frontend UI/UX Skill

Specialized skill for frontend development and design review.

## Use When

- Frontend development
- UI implementation
- UX review
- Component creation
- "frontend", "ui", "ux", "css", "react"

### Component Development
- React/Vue/Angular components

- Responsive design
- Accessibility (a11y)

### Styling
- CSS/SASS/LESS
- Tailwind CSS
- CSS-in-JS

### Review
- Visual hierarchy
- Color contrast

## Design System Integration

```

```

$frontend-ui-ux --review src/components

$frontend-ui-ux --page Dashboard
